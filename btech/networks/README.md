# Computer Networks (Unit 1)

## Introduction

Network is a noun, which means: _Collection of things, communicating with one another_, and in terms of computers,

> A computer network is a collection of devices communicating with one another, via some medium.

Let's understand the point a little bit more detailed. So, the example of network would be something like:

- Your mobile has hotspot enabled and you are connecting to it via your desktop. (A network between mobile and desktop)
- Your printer is connected to the desktop via a cable, or wifi. (A network between computer and printer)
- Your WiFi is connected to some ethernet cable which further is connected to the Modem. (A Network between WiFi and Modem)
- Your tv, desktop, mobile and home devices connected to each other or to internet. (A network between devices and Internet)

Now, the understanding of the network is quite easy, because it is just the devices, communicating with one another.

You connect devices, either via cable or wireless.
