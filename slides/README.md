# Slides to recap the concepts

The slides are written, using markdown and these are formatted as per the _marp_ recommendation.

_Mark_ is a slides generator package, that uses markdown, so go ahead and check it out.

## Installation

Install the `Marp for VS Code` Extension, and you can do everything that marp supports.

Alternatively, you can check out `marp cli` and other tools at: [https://marp.app/](https://marp.app/)
