## Application to learn Programming:

### Web Application

- Role based authentication
- JWT (JSON Web TOken) Authentication
- Seperate API and webpages
- MySql Database/ Mongodb Database
- Redis
- Unit Testing
- API Testing
- File Upload / File Download Functionlity (through frontend)
- CSS and Bootstrap (Styling doesn't matter much.)
- Mobile Responsive (Optional)
- MVC (Modal View Controller) pattern
- Email Verification / SMS based verification

#### Example (A Books App):

Roles:

- User

Permissions:

- User can perform CRUD on books, specific to him
- User can upload cover pictures for the books.
- User can mark a book as public or private.

Goals:

- API + Website
- Simple authentication (Email + Password)
- Email Verification
- MySQL support
- Unit testing
- API testing
